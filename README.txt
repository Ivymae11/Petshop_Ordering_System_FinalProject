PAWVENTORY PET SHOP MANAGEMENT + PRODUCT ORDERING SYSTEM

PHP/Web:
Pet Shop Management System

C# Windows Forms:
Pet Product Ordering System

Database:
SQLite

Connection:
C# connects to PHP through api.php.
Both systems use the same SQLite database.

RUN PHP:
1. Open php-api folder.
2. Run:
   php -S localhost:8000

If php is not found:
   /c/xampp/php/php.exe -S localhost:8000

OPEN PHP ADMIN:
http://localhost:8000/index.php

OPEN C# ORDERING SYSTEM:
csharp\PetProductOrderingApp.sln

DEFAULT PHP ADMIN:
admin@petshop.test
admin123

DEFAULT C# CUSTOMER:
customer@petshop.test
customer123

PHP ADMIN FEATURES:
- Admin login/logout
- Top-navigation retail console
- Product catalog management
- Category management
- Supplier management
- Product stock and reorder monitoring
- Customer account list
- Customer order monitoring
- Order status and payment status update

C# CUSTOMER FEATURES:
- Customer login/logout
- Customer registration
- Product catalog
- Cart and checkout
- Stock-based quantity validation
- Order history
- Order item details
- Cancel pending or approved orders
- Fixed-size window so controls stay visible

API URL:
http://localhost:8000/api.php
